# Gels douche 🚿

## Ingrédients

- base de savon liquide (Marseille, neutre, ...)
- huiles essentielles

## Recette

1. Mélanger la base de savon liquide avec les huiles essentielles
2. Option : ajouter du colorant alimentaire

## Retour d'expérience

Nickel mais attention le savon de Marseille a tendance à former une pâte plus qu'un gel
